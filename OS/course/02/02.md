    cat a.txt | grep foobar

    cut -d: -f6 /etc/passwd|sort|uniq -c|sort -rn

## Stream redirection

    - < : stdin from file
    - > : stdout to file
    - >> : stdout to file (append)
    - 2> : stderr to file (overwrite)
    - 2>> : stderr to file (append)
    - &> : both stdout and stderr
